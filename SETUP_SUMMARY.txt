AGORA (Testar Localmente):
  1. $ npm run check-env
  2. $ npm run dev
  3. Verificar no console (F12) se aponta para localhost:8080

PARA PRODUÇÃO:
  1. Edite .env.production com a URL real da API
  2. $ npm run build
  3. Deploy da pasta dist/ ou configure na plataforma

PLATAFORMAS DE HOSPEDAGEM:
  • Vercel/Netlify: Configure VITE_API_URL no dashboard
  • Servidor próprio: Use .env.production e faça build
  • Docker: Passe VITE_API_URL como build arg
